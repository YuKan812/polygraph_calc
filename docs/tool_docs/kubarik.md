# Инструмент: calculate_kubarik

<!-- keywords: кубарик, кубик, пазл, блок кубиков, куб, нотепад, notepad cube -->

## Когда использовать

Изготовление кубариков — блоков склеенных листов квадратной формы.
Типовые запросы: "кубарик 100 листов", "блок кубиков", "куб из бумаги".

## Состав изделия

Офсетная печать + предварительная резка + промазка клеем + финальная резка.
На одном листе А3 размещается 12 листков кубарика.

## Параметры

- tirazh: тираж кубариков (штук)
- kubelist: листов в одном кубарике (квадратиков)
- listprice: цена бумаги за лист А3 (по умолчанию 2.0)
- cvetnost: цветность офсетной печати
  - "10" = 1+0, "11" = 1+1 (по умолчанию), "20" = 2+0, "22" = 2+2, "21" = 2+1
- pantony: количество пантонов (по умолчанию 0)
- tochnost: "0" приблизительный, "1" точный пантон

## Возвращает

```python
{
    "тип_продукции": "Кубарик/Пазл",
    "тираж": 1000,
    "листов_в_куб": 100,
    "стоимость_печати": 4200.0,
    "стоимость_бумаги": 1680.0,
    "стоимость_предварительной_резки": 540.0,
    "стоимость_промазки": 8400.0,
    "стоимость_финальной_резки": 180.0,
    "количество_листов_а3": 8334,

```

## Пример: кубарик 80 листов, тираж 1000 шт, 1+1

```python
result = calculate_kubarik(
    tirazh=1000,
    kubelist=80,
    listprice=2.0,
    cvetnost="11",
    pantony=0
)

final_answer(
    f"Кубарик {result['листов_в_куб']} листов, тираж {result['тираж']} шт:\n"
    f"Печать: {result['стоимость_печати']:.2f} руб.\n"
    f"Бумага: {result['стоимость_бумаги']:.2f} руб.\n"
    f"Резка (предв. + фин.): "
    f"{result['стоимость_предварительной_резки'] + result['стоимость_финальной_резки']:.2f} руб.\n"
    f"Промазка клеем: {result['стоимость_промазки']:.2f} руб.\n"
    f"Итого: {result['общая_стоимость']:.2f} руб. / "
    f"{result['цена_за_единицу']:.2f} руб. за шт."
)
```
