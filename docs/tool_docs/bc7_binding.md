# Инструмент: calculate_bc7_binding

<!-- keywords: 7БЦ, твёрдый переплёт, брошюровка, книга, блок, тетрадь, шитьё, форзац, бумвинил, картон, кругление, корешок, КШС, переплёт -->

## Когда использовать

Расчёт твёрдого переплёта (7БЦ) для книг, ежедневников, фотоальбомов.
Типовые запросы: "твёрдый переплёт", "7БЦ", "брошюровка книги", "ежедневник".

## КРИТИЧНО: полосы = страницы = одно и то же!

```
"40 страниц" → polosy=40
"40 полос"   → polosy=40
"32 страницы (64 полосы)" → это ОШИБКА в формулировке! Правильно: polosy=32
```

НЕ удваивать и не делить — брать число как есть.

## Параметры

- tirazh: тираж (штук)
- format_val: формат блока — "A4", "A5", "A3", "A6"
- grammazh: плотность бумаги г/м²:
  - < 100 г/м² → 24 страницы в тетради
  - 100–129 г/м² → 16 страниц в тетради
  - ≥ 130 г/м² → 8 страниц в тетради
- polosy: ОБЩЕЕ число полос (страниц) в блоке
- rounding: кругление корешка — True/False (по умолчанию False)
- cardboard: толщина картона обложки — "1.5", "1.75", "2.0", "2.5" мм
- cover_material: материал обложки
  - "paper" — бумага (по умолчанию, печать считается)
  - "bumvinil" — бумвинил (печать НЕ считается, делается у подрядчика)
- cover_color: цветность печати обложки — "40", "44", "41", "0" (без печати)
- manager: ключ менеджера (опционально)
- notes: заметки (опционально)

## Ключи менеджеров

vladimirova, dmitriev, dustova, minenkov, kursanov, pokrovskaja,
salnikova, hokhlova, feoktistova, chistyakova

## Материал обложки — правила

- По умолчанию: cover_material="paper"
- Явно указано "бумвинил" → cover_material="bumvinil"
- При бумвиниле: cover_color можно указать, но стоимость печати НЕ включается в расчёт

## Пример: ежедневник А5, 320 стр, бумага 80 г/м², бумвинил

```python
result = calculate_bc7_binding(
    tirazh=100,
    format_val="A5",
    grammazh=80,       # < 100 → 24 стр/тетрадь → ceil(320/24) = 14 тетрадей
    polosy=320,        # 320 страниц = 320 полос (одно и то же!)
    rounding=True,     # кругление корешка для толстой книги
    cardboard="2.0",
    cover_material="bumvinil",
    cover_color="0",   # без печати (у подрядчика)
    manager="chistyakova",
    notes="Ежедневник на 2025 год"
)

final_answer(
    f"Брошюровка 7БЦ, {result['тираж']} шт:\n"
    f"Стоимость: {result['полная_стоимость']}\n"
    f"За единицу: {result['цена_за_единицу']}\n"
    f"Сборка КШС: {result['сборка_КШС_итог']} / {result['сборка_КШС_ед']} за шт.\n"
    f"PDF: {result['pdf_path']}"
)
```

## Пример: книга А4, 192 стр, меловая бумага 130 г/м², бумага обложки 4+1

```python
result = calculate_bc7_binding(
    tirazh=500,
    format_val="A4",
    grammazh=130,      # ≥ 130 → 8 стр/тетрадь → ceil(192/8) = 24 тетради
    polosy=192,
    rounding=False,
    cardboard="2.5",
    cover_material="paper",  # бумага — печать рассчитывается
    cover_color="41",        # 4+1
    manager="vladimirova"
)

final_answer(
    f"Брошюровка 7БЦ А4, {result['тираж']} шт:\n"
    f"Стоимость: {result['полная_стоимость']}\n"
    f"За единицу: {result['цена_за_единицу']}\n"
    f"PDF: {result['pdf_path']}"
)
```
