# Инструмент: calculate_bc7_binding
# Расчёт стоимости брошюровки 7БЦ (твёрдый переплёт)

## Назначение
Рассчитывает стоимость сборки книги/брошюры в твёрдый переплёт (7БЦ):
фальцовка/подъём, шитьё, форзацы, картон, крышка, вставка, кругление.

---

## ⚠️ КРИТИЧНЫЕ ПРАВИЛА РАЗБОРА ЗАПРОСА

### 1. `cover_material` — материал КРЫШКИ обложки
Принимает только два значения:
- `"paper"` — **бумага/картон** (меловка, офсет, дизайнерская бумага)
- `"bumvinil"` — **бумвинил** (виниловое покрытие, тканевая основа)

**УСТАНОВИТЬ `cover_material="paper"` если в запросе:**
- меловка (любой граммаж: 150, 250, 300 г/м²)
- офсет на обложку
- дизайнерская бумага
- картон
- "обложка 4+1", "4+0", "4+4" — цветность описывает ПЕЧАТЬ, не материал

**УСТАНОВИТЬ `cover_material="bumvinil"` ТОЛЬКО если написано:**
- "бумвинил" явно
- "bumvinil"

### 2. "Плёнка" ≠ бумвинил — ЧАСТАЯ ОШИБКА!
"Плёнка глянцевая", "плёнка матовая", "ламинация плёнкой 75 мкм" —
это ОТДЕЛКА ПОВЕРХНОСТИ, а НЕ материал обложки.
При наличии слова "плёнка" в запросе:
- `cover_material="paper"` (обложка остаётся бумагой/меловкой)
- Информацию о плёнке вносить в `notes`

**Примеры:**
| Запрос | cover_material | notes |
|---|---|---|
| "меловка 150 г/м², плёнка глянцевая" | `"paper"` | "плёнка глянцевая" |
| "обложка бумвинил" | `"bumvinil"` | — |
| "обложка 4+1, ламинация 75 мкм" | `"paper"` | "ламинация 75 мкм" |
| "корешок прямой, плёнка глянцевая" | `"paper"` | "корешок прямой, плёнка глянцевая" |

### 3. `rounding` — кругление корешка
- `rounding=True` — "кругление", "округлый корешок"
- `rounding=False` — "прямой корешок", "без кругления", **по умолчанию False**
- "корешок прямой" → `rounding=False` (НЕ True!)

### 4. `cardboard` — толщина картона
По умолчанию `"2.0"` если явно не указана другая толщина.
Допустимые значения: `"1.5"`, `"1.75"`, `"2.0"`, `"2.5"`

### 5. Имя менеджера — брать из подписи письма
Если запрос содержит подпись вида "С уважением, Имя Фамилия",
"Всегда рада сотрудничеству, Лидия Чистякова" и т.п.:
- Извлечь имя и передать в `manager`
- Поддерживаемые ключи: vladimirova, dmitriev, dustova, minenkov,
  kursanov, pokrovskaja, salnikova, hokhlova, feoktistova, **chistyakova**
- "Лидия Чистякова", "Чистякова", "chistyakova" → `manager="chistyakova"`
- Если имя не распознано — передать как есть (кириллицей)

### 6. Форзацы
Форзацы — всегда часть книжного блока. Слова "форзац непечатный", 
"форзац офсет 120 г/м²" — это описание материала для notes, 
не влияет на параметры `calculate_bc7_binding`.

### 7. Цветность обложки `cover_color`
- "4+1" → `cover_color="41"`
- "4+0" → `cover_color="40"`  
- "без печати", "непечатная" → `cover_color="0"`

## Параметры функции

```python
calculate_bc7_binding(
    tirazh: int,          # тираж штук
    format_val: str,      # "A4", "A5", "A3", "A6"
    grammazh: int,        # плотность бумаги блока г/м²
    polosy: int,          # страниц (= полос) — ОДНО И ТО ЖЕ
    rounding: bool,       # кругление корешка (False по умолчанию)
    cardboard: str,       # "1.5","1.75","2.0","2.5" (по умолч. "2.0")
    cover_material: str,  # "paper" или "bumvinil"
    cover_color: str,     # "41","40","44","0" и др.
    manager: str,         # ключ или имя менеджера
    notes: str,           # заметки о материалах, плёнке и др.
)
```

**Страницы = Полосы — одно и то же!**
`polosy=240` означает 240 страниц (полос). НЕ удваивать.

**Граммаж определяет тетради:**
- < 100 г/м² → 24 стр/тетрадь
- 100–129 г/м² → 16 стр/тетрадь  
- ≥ 130 г/м² → 8 стр/тетрадь

**Формат:** если указан нестандартный (напр. 205×290 мм) — использовать ближайший:
- 205×290 ≈ A4 → `format_val="A4"`

---

## Эталонный пример разбора

**Запрос:**
> Формат 205×290 мм, объём 240 с., переплёт 7БЦ, бумага офсет 80 г/м² 1+1,
> форзацы офсет 120 г/м² непечатные, обложка меловка 150 г/м² 4+1,
> корешок прямой, плёнка глянцевая, тираж 20 экз.
> Всегда рада сотрудничеству, Лидия Чистякова

**Разбор:**
| Параметр | Значение | Обоснование |
|---|---|---|
| `tirazh` | 20 | явно указан |
| `format_val` | "A4" | 205×290 ≈ A4 |
| `grammazh` | 80 | офсет 80 г/м² |
| `polosy` | 240 | 240 страниц |
| `rounding` | False | "корешок прямой" |
| `cardboard` | "2.0" | не указана → по умолчанию |
| `cover_material` | **"paper"** | меловка = бумага, НЕ бумвинил |
| `cover_color` | "41" | "4+1" |
| `manager` | "chistyakova" | из подписи "Лидия Чистякова" |
| `notes` | "Обложка меловка 150 г/м² 4+1, плёнка глянцевая. Форзацы непечатные, офсет 120 г/м²." | детали материалов |

```python
result = calculate_bc7_binding(
    tirazh=20,
    format_val="A4",
    grammazh=80,
    polosy=240,
    rounding=False,
    cardboard="2.0",
    cover_material="paper",
    cover_color="41",
    manager="chistyakova",
    notes="Обложка меловка 150 г/м² 4+1, плёнка глянцевая. Форзацы непечатные, офсет 120 г/м²."
)
```

---

## Что НЕ рассчитывает этот инструмент
- Стоимость печати обложки и блока (отдельные инструменты)
- Стоимость ламинации/плёнки (отдельный инструмент)
- Стоимость бумаги и форзацев
Эти параметры фиксируются в `notes` для менеджера.
