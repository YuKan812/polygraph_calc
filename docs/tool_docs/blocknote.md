# Инструмент: calculate_blocknote

<!-- keywords: блокнот, тетрадь, записная книжка, пружина, спираль, блок листов, обложка, подложка, резка, счет листов -->

## Когда использовать

Изготовление блокнотов и тетрадей с металлической пружиной.
Типовые запросы: "блокнот А5 с пружиной", "тетрадь 50 листов", "записная книжка с ламинацией".

## Состав изделия

1. Блок листов — офсетная печать
2. Обложка (картон) — цифровая печать
3. Подложка (картон, опционально с печатью и ламинацией)
4. Пружина металлическая
5. Ламинация обложки и/или подложки (опционально)

## КРИТИЧНО: результат

```python
result = calculate_blocknote(...)
# Передавать ТОЛЬКО result['ответ'] в final_answer — НЕ переформатировать!
final_answer(result['ответ'])
```

## Параметры

### Основные
- tirazh: тираж блокнотов (штук)
- list_block: листов в блоке (рекомендуется 40–50)
- format_val: формат — A6=8, A5=4, A4=2, A3=1
- storona: сторона пружины — "1" (короткая), "2" (длинная)

### Печать блока (офсет)
- cvetnost: цветность блока
  - "10" = 1+0, "11" = 1+1, "20" = 2+0, "22" = 2+2, "21" = 2+1
- pantony: количество пантонов (по умолчанию 0)
- tochnost: "0" приблизительный, "1" точный подбор пантона
- list_price: цена бумаги блока за лист А3 (по умолчанию 2.0)

### Резка
- rezka: "1" — один рез, "2" — два реза

### Пружина
- pruzhina: размер пружины 0–5:
  - 0 = 1/4 d 6,4 мм (до 20 листов)
  - 1 = 5/16 d 7,9 мм (20–40 листов) ← рекомендуется для 40–50 л.
  - 2 = 3/8 d 9,5 мм (50–80 листов)
  - 3 = 7/16 d 11,1 мм (80–100 листов)
  - 4 = 1/2 d 12,7 мм
  - 5 = 9/16 d 14,3 мм

### Обложка (цифровая печать)
- cover_price: цена картона обложки за лист А3 (по умолчанию 5.0)
- cover_color_count: цветность обложки
  - "40" = 4+0, "44" = 4+4, "10" = 1+0, "11" = 1+1
- lam_quality: ламинация обложки
  - "1" = нет, "2" = глянцевая, "3" = матовая, "4" = плёнка 75, "5" = плёнка 125
- lam_sides: "1+0" (по умолчанию) или "1+1"

### Подложка (опционально)
- backing_color_count: цветность подложки — "0" = без печати (по умолчанию)
- backing_lam_quality: ламинация подложки — "1" = нет (по умолчанию)
- backing_lam_sides: "1+0" (по умолчанию)

## Типовые значения пружины по количеству листов

| Листов в блоке | Рекомендуемая пружина |
|---|---|
| до 20 | pruzhina=0 |
| 20–40 | pruzhina=1 |
| 40–80 | pruzhina=2 |
| 80–100 | pruzhina=3 |

## Пример: блокнот А5, 50 листов, 1+1, ламинация глянец

```python
result = calculate_blocknote(
    tirazh=500,
    list_block=50,
    format_val=4,           # А5
    storona="1",            # короткая сторона
    cvetnost="11",          # 1+1 ч/б офсет
    pruzhina=2,             # 3/8 d 9,5 мм для 50 листов
    cover_price=5.0,
    cover_color_count="44", # обложка 4+4
    lam_quality="2",        # глянцевая ламинация обложки
    lam_sides="1+0"         # снаружи
)
final_answer(result['ответ'])
```

## Пример: блокнот А4, 100 листов, без ламинации

```python
result = calculate_blocknote(
    tirazh=200,
    list_block=100,
    format_val=2,           # А4
    storona="1",
    cvetnost="11",
    pruzhina=3,             # 7/16 для 100 листов
    cover_color_count="40", # обложка 4+0
    lam_quality="1"         # без ламинации
)
final_answer(result['ответ'])
```
