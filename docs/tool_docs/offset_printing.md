# Инструмент: calculate_offset_printing

<!-- keywords: офсетная печать, офсет, 1+0, 1+1, 2+0, 2+2, пантон, форма, приладка, тираж офсет -->

## Когда использовать

Офсетная печать — большие тиражи (от 300–500 экз.), экономичнее цифры при больших объёмах.
Типовые запросы: "офсетная печать", "офсет 1000 экз.", "печать с пантоном".

## ВАЖНО: "офсетная бумага" ≠ "офсетная печать"

- "цифровая печать на офсетной бумаге 80 г/м²" → инструмент calculate_DIGITAL_printing, бумага офсетная
- "офсетная печать 1+1" → инструмент calculate_OFFSET_printing

## Обязательный порядок

1. calculate_layout → получить items_per_sheet
2. calculate_offset_printing(tirazh, items_per_sheet, ...)
3. "количество_листов_а3" передать в calculate_lamination если нужна ламинация

## Параметры

- tirazh: количество экземпляров изделий
- items_per_sheet: из результата calculate_layout
- colorness: цветность печати
  - "10" — 1+0 (одна краска одна сторона)
  - "11" — 1+1 (одна краска две стороны)
  - "20" — 2+0 (две краски одна сторона)
  - "22" — 2+2 (две краски две стороны)
  - "21" — 2+1 (асимметричная)
- paper_price: цена листа А3 (по умолчанию 2.0)
- material: "1" — бумага (по умолчанию), "2" — картон
- pantone_count: количество пантонов (по умолчанию 0)
- pantone_accuracy: "0" — приблизительный, "1" — точный подбор

## Интерпретация цветности

- "1+0", "1 краска", "чёрная" → colorness="10"
- "1+1", "ч/б двусторонняя офсет" → colorness="11"
- "2+0", "2 краски одна сторона" → colorness="20"
- "2+2", "2 краски обе стороны" → colorness="22"

## Возвращает

```python
{
    "тип_печати": "Офсетная",
    "стоимость_печати": 3200.0,
    "стоимость_бумаги": 880.0,
    "общая_стоимость": 4080.0,
    "количество_листов_а3": 440.0,  # ← передавать в calculate_lamination
    "количество_форм": 2,
    "цена_за_единицу": 4.08
}
```

## Пример: 1000 экз. А5, 1+1, офсетная бумага 80 г/м²

```python
layout = calculate_layout(piece_width=148, piece_height=210)
items = layout["items_per_sheet"]  # → 4

result = calculate_offset_printing(
    tirazh=1000,
    items_per_sheet=items,
    colorness="11",
    paper_price=2.0,
    material="1"
)
sheets = result["количество_листов_а3"]
```
